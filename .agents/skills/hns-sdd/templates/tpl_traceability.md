## 矩阵说明

| 字段 | 含义 |
|------|------|
| `TASK ID` | 任务编号，对应 `tasks.md` |
| `REQ / AC` | 关联需求编号和验收标准编号 |
| `状态` | `待开始` / `进行中` / `done` / `BLOCKED` / `WONTFIX` |
| `实现文件` | 主要实现代码的文件路径（完成后填写） |
| `测试文件` | 对应测试文件路径（完成后填写） |
| `AC 报告` | 验收矩阵文件路径，如 `docs/spec/ac/TASK-001-AC.md` |
| `完成日期` | 任务关闭日期 |
